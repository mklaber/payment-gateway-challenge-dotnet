## Pre-Requisites

* [.NET 8.0][net8]
* Docker

This project was provided with a bank simulator. Start it before running the app and tests:

```bash
docker compose up
```

You can verify the simulator is running by navigating to [http://localhost:2525/][sim-lh] in your browser.
